# GenAI-FAQs-Ethics

# Project Title:
GenAI FAQs & Ethical Use of AI

# Short Overview:
This project is a visually appealing web-based FAQ and educational resource platform focused on Generative AI, its practical uses, ethical guidelines, and real-world analogies. It aims to help beginners in school, research, and daily life understand what Generative AI is, its strengths, limitations, and responsible use.

# Instructions on How to Run the Project:
Prerequisites: A modern web browser (Chrome, Firefox, Edge, Safari).
Installation: Clone or download the project files to your local machine.
Running: Open the main HTML file (index.html) in your preferred browser. No server setup needed as this is a static site.
Usage: Navigate through the sections via the navbar to explore FAQs, ethical guidelines, and analogy explanations about AI. The contact form alerts a thank-you message on submission.

# Frameworks/Libraries Used: 
Bootstrap 5.3 for responsive design and UI components
Bootstrap Icons for iconography
Simple JavaScript for form submission alert
Google Fonts or system fonts for typography
